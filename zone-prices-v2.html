📦 Woop Zone Pricing Analysis v2

Historical delivery rates by zone • Base rate ex-fuel ex-GST • Corrected methodology

⚠️ Methodology Notes

  • 2015-2020: rawbaseamount was NULL. Base rate = ucjbamount - fuelsurchargeamount. All jobs are standalone (type 1).
  • 2021: Transitional — uses rawbaseamount where populated, otherwise falls back to ucjbamount - fuelsurchargeamount.
  • 2022-2023: Uses rawbaseamount directly. All jobs still standalone type 1.
  • 2024: Mix of standalone type-1 jobs (speed 95) and new parent/child model (type 20, speed 94/95). Both included; parent jobs excluded where ucjbid = parentid.
  • 2025: Fully parent/child model. Only type-20 delivery children (speed 94/95) used — these represent the actual delivery leg rate.
  • Thresholds: Rates below $4.00 or above $200.00 excluded (pickup legs, errors, outliers).
  • Sunday rates shown separately where data exists.

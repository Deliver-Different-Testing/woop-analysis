Methodology:
  • MFV = 6.3% × base revenue per run (driver receives 100%)
  • Extra fuel cost = (pump price − $1.86) × litres consumed. MFV only needs to cover cost ABOVE the $1.86/L base.
  • Return to base: All Woop runs are return trips (pickup return boxes). KMs doubled.
  • Consumption: 13 L/100km default (real-world loaded urban). Adjustable.
  • Data: 126 Auckland Sunday runs with KM data from tblBulkRun, Nov 2025 – Feb 2026.
  • MFV rate source: tblFuelSurcharge — 6.3% at $2.584/L pump price (Mar 2026).

Methodology

  • Distance calculation: Haversine great-circle distance between depot (pickup) and each delivery point, sequenced using a nearest-neighbour TSP heuristic (depot → nearest unvisited → ... → depot).
  • Road multiplier: 1.27× applied to straight-line distance to estimate actual road distance (standard NZ urban road factor).
  • Run grouping: Each unique combination of run name + date constitutes a distinct run (run names repeat weekly).
  • Revenue: Sum of ucjbAmount for all non-void deliveries in the run (incl. GST, fuel surcharge).
  • Client codes: WOOP, WOOPH, WOOPW, WOOPC (all Woop bulk delivery operations).
  • City classification: Based on run name prefix (AW=Auckland, WW=Wellington, CW=Christchurch, HW=Hamilton, TW=Tauranga), with depot coordinates as fallback.
  • Data source: Railway PostgreSQL (tucjobarchive), 989,007 deliveries across 37,625 runs.
